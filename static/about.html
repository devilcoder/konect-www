About the KONECT Project

The KONECT project is run at the University of Namur by Jérôme Kunegis.

Citing the KONECT Project

When referring to the KONECT project, cite the following paper: 
@inproceedings{konect,
	title = {{KONECT} -- {The} {Koblenz} {Network} {Collection}},
	author = {J\'{e}r\^{o}me Kunegis},
	year = {2013},
	booktitle = {Proc. Int. Conf. on World Wide Web Companion},
	pages = {1343--1350},
	url = {http://dl.acm.org/citation.cfm?id=2488173},
}
An up-to-date list of papers citing KONECT is given by Google Scholar.
